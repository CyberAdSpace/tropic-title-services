# Tropic Title Services

Modern website for Tropic Title Services — Florida's trusted family-owned title company since 2005.

## Pages

- **Home** (`index.html`) — Hero, welcome, services preview, why us, CTA
- **Meet the Team** (`team.html`) — About section, staff profiles
- **Why Choose Us?** (`services.html`) — Detailed services, instant estimates
- **Contact Us** (`contact.html`) — Contact form, phone/email/location info
- **Title Order Form** (`order.html`) — Full transaction order form

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter + Playfair Display)
- CSS Custom Properties (design tokens)
- IntersectionObserver for scroll animations
- Fully responsive (mobile-first)
- No build step required — static files only

## Development

Simply open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
```

## Deployment

Deploy to any static hosting (Netlify, Vercel, GitHub Pages, etc.).
